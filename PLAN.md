# 작전 바이브 시뮬 - 소부대 통신 전술 훈련 시뮬레이터

## 프로젝트 개요
- **프로젝트 명** Signal-Fog ( 작전 바이브 시뮬 / 무전 오류 훈련 시뮬레이터 )  
- **슬로건**: "작전간의 워게임을 재미있게 하며 통신 축선을 구축하라 – 분대장님, 명령 하달!"  
- **팀 구성**: 팀명 - LNG : 2인 팀 (상병 김응태 + 상병 장호영)  
- **대회**: 2026 제15회 육군창업경진대회 예선 출품  
- **형태**: 웹 기반 멀티플레이 턴제 탑뷰 2D 전략 시뮬레이션 (Phaser.js + TensorFlow.js CDN)  
- **목표**: 군 소부대 작전 훈련 중 발생하는 **무전기(통신) 오류·판단 미스·정보 왜곡**을 재미있고 반복 학습 가능한 시뮬레이션으로 해결 → 실전 오류율 감소 + 장병 훈련 효율화 + 피로가 해소되는 재미있는 워게임 가능

## 문제 정의 (군 생활 Pain Point)
소부대(분대·소대) 작전 및 KCTC 훈련 시 가장 빈번한 실패 원인 중 하나:
- 무전기 통신 지연 / 노이즈 / 오청 → 수신호·판단 오류 발생
- 실제 훈련 전 미리 연습할 방법 부족 → 실전에서 반복 실수
- 결과: 작전 실패, 시간 지연, 안전 문제, 전투력 저하

→ 모든 육군 장병(특히 전방·전투병과)이 공감하는 **실전 통신·판단 고충**  
(역대 수상작 중 모션인식 장갑 '플랙사인'처럼 통신/수신호 관련 아이템이 최우수상 받은 사례 있음)

## 솔루션: 작전 바이브 시뮬레이터
- **핵심 기능**
  - 턴제 탑뷰 2D 지도 (8x8 그리드 기본, 확장 가능)
  - 개인 스탯 커스텀 (체력 / 상황판단 / 사격 / 이동력 등) → 장병별 특성 반영
  - 무전기 채팅 시뮬 (명령 입력 → 팀원 판단 스탯 따라 성공/오류 랜덤 발생)
  - 의도적 노이즈·도청·정보 왜곡 구현 (예: "3,4 이동" → "5,6으로 오해" 등)
  - 레이더(미니맵) + 포그 오브 워 → 정보 불완전성 재현
  - 한 턴 행동 제한 (이동 + 행동 1~2회) → 실제 작전 리듬감

- **이중용도 (군 + 민간)**
  - **군 내**: 소부대 훈련 보조 도구 (KCTC 전 미리 오류 연습 → 오류율 20~30% 감소 기대)
  - **민간 확장**: Airsoft/서바이벌 클럽 전술 훈련, ROTC·예비군 교육 툴, 기업 팀빌딩·리더십 전략 시뮬, e스포츠 전략 게임 시장 진입

- **개발 철학**: 사지방 제약 100% 고려 (설치 0, GitHub Pages 배포, 브라우저 실행)

## 기술 스택 (현재 계획)
- **Frontend**: HTML + Phaser.js 3.90.0 (CDN) – 2D 게임 엔진
- **AI/로직**: JavaScript 내장 + TensorFlow.js (CDN, 간단 판단 오류 모델 가능)
- **배포**: GitHub Pages (무료, 사지방 접근 용이)
- **협업**: GitHub private repo (2인 무제한 collaborators)
- **추가 고려**: 
  - 사운드: Howler.js 또는 무료 웹 사운드 (군 무전기 효과음)
  - 에셋: 무료 픽셀 아트 (OpenGameArt.org 등) + 군 지도 텍스처
  - 나중 확장: WebRTC (멀티플레이 데모용, 본선 이후)

## 개발 로드맵
1. **현재 ~ 3월 말**: 기본 멀티플레이 프로토타입 완성 (지도, 유닛 이동, 채팅 오류 로직)
2. **4월 초**: 육군 바이브 UI 포장 (군청색 톤, 군용 폰트, 명령 하달 텍스트 애니메이션) + 1~2분 데모 영상 촬영
3. **4월 10일 전**: 예선 접수 (유튜브 링크 첨부)
4. **본선 진출 시**: 멀티 협력 + AI 난이도 자동 조절 추가
