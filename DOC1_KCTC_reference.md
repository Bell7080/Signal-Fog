# 문서 1 — KCTC 과학화 전투훈련 고증 참고자료

> 분류: 참고 자료 (READ-ONLY 베이스라인)
> 용도: Signal-Fog 기획·개발 전 과정에서 고증 판단 기준으로 사용
> 수정 시: 반드시 출처 명기 후 하단 변경 이력에 기록

---

## 1. KCTC 기본 제원

| 항목 | 내용 |
|------|------|
| 정식 명칭 | 육군과학화전투훈련단 (Korea Combat Training Center) |
| 위치 | 강원도 인제군 |
| 훈련장 면적 | 약 250km² |
| 기준 훈련 단위 | 대대급 (700~900명) / 일부 여단급 |
| 훈련 기간 | 14~21일 (준비~평가 전 과정) |
| 훈련 주기 | 부대별 2~3년 1회 순환 |
| 총 참여 인원 | 훈련부대 + 대항군 + 통제관 + 지원 합산 2,000~3,000명 |
| 벤치마크 | 미군 NTC (National Training Center) |

---

## 2. 편성 구조

### 2-1. 훈련부대 (Blue Force)

| 제대 | 구성 |
|------|------|
| 대대 본부 | 대대장, 부대대장, 작전장교, 정보장교 |
| 본부중대 | 통신·의무·보급·공병 배속 |
| 보병중대 | 3~4개 중대, 중대장·소대장·분대장·병사 |
| 직할 화기 | 박격포소대, 대전차소대 |

### 2-2. 대항군 (OPFOR / Red Force)

- 상설 전문 대항부대 운용 (연중 훈련으로 전투력 유지)
- 지형 숙지도·전술 숙련도 훈련부대 대비 압도적 우위
- 기습·매복·위장 침투 특화
- 실화: 대항군 요원이 아군 복장 위장 → 신병 접근 → 배치 정보 획득 후 사살

### 2-3. 통제관 (OC/T)

- 소대~중대 단위 전담 배치
- 교전 판정·규칙 감독·데이터 수집·AAR 주도

---

## 3. 핵심 장비 체계

### 3-1. MILES

| 구성요소 | 기능 |
|----------|------|
| 레이저 발사기 | 총기·포에 부착, 발사 시 레이저 송출 |
| 피탄 감지기 | 조끼·헬멧·차량 부착 센서 |
| 사상 판정기 | 피격 시 버저음 + LED → 즉사/부상/행동불능 |
| 차량용 MILES | 장갑차·전차 별도 장착 |

무기별 레이저 코드 분리: 소총 / 기관총 / 박격포 / 대전차화기 / 포병

### 3-2. 수류탄 모의장비

- 모의 수류탄 (폭음 + 섬광)
- 살상반경 / 파편반경 구분
- 통제관이 반경 내 피해 직접 판정

### 3-3. 포병·박격포 모의

- 실사격 불가 → 좌표 보고 + 통제관 판정
- 실화: 작전병 좌표 오류 → 아군 진지 피격 / 적 대대장 격파 (양쪽 사례 모두 존재)

### 3-4. 위치추적 시스템

- 전 인원·장비 GPS 추적 장치
- 실시간 통제본부 모니터링
- 이동 경로·교전 위치 전부 기록

## 4. 훈련 단계별 진행

| 단계 | 기간 | 내용 |
|------|------|------|
| 준비·이동 | D-7 ~ D-3 | 장비 수령, MILES 장착, ROE 교육, 작전계획 수립 |
| 방어훈련 | D-1 ~ D+4 | 진지 구축 → 적 공격 격퇴 |
| 공격훈련 | D+5 ~ D+8 | 기동 → 돌파 → 목표 탈취 |
| 연속작전 | D+7 ~ D+10 | 72~96시간 연속 운용 (수면 제한) |
| 평가·AAR | D+11 ~ D+14 | 데이터 분석, 제대별 AAR, 개선과제 부여 |

## 5. 생존 스탯 — 실제 훈련 고증

### 5-1. 피로도

| 상황 | 실제 영향 |
|------|---------|
| 72~96시간 연속작전 | 판단력 저하, 사격 정확도 감소, 명령 오인 빈도 증가 |
| 수면 4시간 이하 지속 | 경계 실패, 지휘관 결심 지연 |
| 야간 작전 후 주간 연속 | 누적 피로 → 기동 속도 저하 |

**개인차 근거**: 체력 우수 장병이 피로 상황에서도 포격 유도 임무 완수한 사례 KCTC AAR에 반복 등장. 체력이 전투 지속성에 직결된다는 평가 공식 기록됨.

### 5-2. 식수·배고픔

| 상황 | 실제 영향 |
|------|---------|
| 급수차 피격·파괴 | 식수 공급 중단 → 전투력 저하 판정 |
| 급수대 점령당함 | 보급선 차단 |
| 전투식량 미수령 3~4시간 | 행동력 저하 |

### 5-3. 수면

| 상황 | 실제 영향 |
|------|---------|
| 야간 경계 근무 후 | 익일 전투력 저하 |
| 연속작전 48시간 초과 | 환각·판단 오류 사례 보고됨 |
| 수면 확보 성공 시 | 다음 작전 전투력 부분 회복 |

## 6. 통신 실패 — 실제 KCTC 사례 기록

| 사례 | 원인 | 결과 |
|------|------|------|
| 포격 좌표 오류 | 작전병 무전 오청 | 아군 진지 피격 또는 역으로 적 지휘부 격파 |
| 무전기 피탈 | 수색대원 항복 시 무전기 탈취 | 대항군이 아군 채널로 역공격 |
| 통신팀 전멸 | 복구 작업 중 유탄 피격 | 전 제대 통신 두절, 지휘체계 붕괴 |
| 위장 침투 | 대항군 아군 복장 착용 | 신병에게 접근·정보 획득 후 사살 |
| 채널 혼선 | 동일 주파수 중복 사용 | 명령 중첩·오인 |
| 계곡 진입 | 지형 통신 음영 | 소대 단위 완전 통신 두절 |
| 배터리 고갈 | 장시간 운용 중 교체 미실시 | 무전기 작동 불능, 예외처리 판정 |
| 내부자 제보 | 병사가 대항군에 아군 정보 제공 | 아군 배치 노출, 기습 당함 |

## 7. 보급 체계

### 7-1. 5대 보급 자원

| 자원 | 고갈 시 실제 영향 |
|------|----------------|
| 탄약 | 해당 화기 전투 불가 |
| 유류 | 차량·장갑차 이동 불가 |
| 식수 | 전투력 점진적 저하 |
| 의무 물자 | 부상자 처치 불가 → 전사 전환 |
| 통신 배터리 | 무전기 불통 → 지휘 두절 |

### 7-2. 보급선 차단

- 보급 차량 경로 상 적 존재 시 이동 불가
- 차량 피격·파괴 시 해당 물자 전량 소멸
- 급수대: 맵 고정 위치, 적 점령 시 식수 차단

## 9. 사상 판정 및 인수인계 체계

### 9-1. 사상 3단계

| 판정 | 즉각 효과 | 후속 처리 |
|------|---------|---------|
| 즉사 | 즉시 전투 이탈 | 인수인계 발동 |
| 부상(행동불능) | 이동·전투 불가 | 의무병 후송 시 복귀 가능 |
| 경상 | 전투력 부분 저하 | 의무병 처치 시 회복 |

### 9-2. 지휘관 사망 시 인수인계 체계 (KCTC 실제 운용)

```
대대장 사망 → 부대대장 즉시 승계 (1턴 이내)
   └─ 부대대장도 사망 → 선임 중대장이 대대장 직무 대행
       └─ 인수인계 미선언 시 → 전 제대 독립행동 모드

중대장 사망 → 선임 소대장 즉시 승계
   └─ 2턴 초과 미승계 → 중대 지휘 두절 판정

소대장 사망 → 선임 분대장 승계
   └─ 미승계 → 분대별 개별행동만 가능
```

### 9-3. 의무 후송 체계

| 단계 | 위치 | 처리 |
|------|------|------|
| 1선 | 전투 현장 | 전우 응급처치·후송 |
| 2선 | 대대 의무반 | 의무병 처치, 후송 판정 |
| 3선 | 후방 의무대 | 중증 → 전투력 복귀 불가 |

- 후송 지연 2턴 초과 시 부상 → 전사 전환
- 이송 중 의무병도 피격 대상
