## 2. 핵심 컨셉

> "통신이 살아있어야 작전이 살아있다 — 분대장님, 명령 하달!"

KCTC 훈련에서 가장 빈번한 실패 원인인 **통신 오류·판단 미스**를 게임의 핵심 메커니즘으로 설계. 단순 전투 게임이 아니라 명령→전달→판단→실행 전 과정에서 오류가 발생하는 구조.

---

## 3. 턴 시스템 — 지휘계통 우선순위 실시간 턴제

### 3-1. 설계 원칙

KCTC 및 실제 군 지휘체계 고증: 명령은 반드시 상위 제대에서 하위 제대 순으로 흐른다. 이를 턴 실행 우선순위에 그대로 반영.

```
[턴 입력 페이즈] — 전 플레이어 동시 입력 (제한 시간 내)
   └─ 지휘관부터 병사까지 각자 행동 입력
   └─ 입력 시간: 60초 (추후 조정 가능)

[턴 실행 페이즈] — 지휘계통 우선순위 순서로 자동 실행
   1순위: 중대장 (명령 하달·화력 지원 요청)
   2순위: 소대장 (소대 기동 명령)
   3순위: 무전병 (통신망 유지·채널 관리)
   4순위: 분대장 (분대 전술 행동)
   5순위: 특기병 (공병·의무병·보급관)
   6순위: 병사 (개별 전투 행동)
```

### 3-2. 통신 오류의 턴 영향

- 중대장 명령이 통신 두절 시 → 소대장은 이전 명령 또는 독립 판단으로만 행동
- 무전병 사망 시 → 해당 턴 통신 관련 행동 전부 불가
- 이 구조가 "지휘관부터 보호해야 한다"는 전술 판단을 자연스럽게 유도

### 3-3. 턴 제한 시간

| 페이즈 | 시간 | 비고 |
|--------|------|------|
| 입력 페이즈 | 60초 | 미입력 시 대기(Hold) 자동 처리 |
| 실행 페이즈 | 자동 | 애니메이션 속도로 순차 실행 |
| 인수인계 페이즈 | 30초 | 지휘관 사망 직후 발동 |

### 4-3. 야간 페이즈 지형 변화

- 모든 시야 범위 -2타일
- 개활지 이동 시 위치 노출 확률 +30% (소음)
- 수면을 확보한 유닛: 야간 페이즈 시야 정상 유지 (야간투시경 보유 판정)
- 수면 부족 유닛: 시야 추가 -1, 오청 확률 +10%

## 5. 역할 편성 시스템 — 중대급 (30~50명)

### 5-1. Blue Force 편성표

| 역할 | 인원 | AP | 주요 기능 | 고증 근거 |
|------|------|-----|---------|---------|
| **중대장** | 1 | 3 | 전체 명령 하달, 포병·항공 지원 요청 | KCTC 중대 지휘관 |
| **부중대장** | 1 | 3 | 중대장 사망 시 즉시 승계 | 인수인계 체계 |
| **소대장** | 3 | 4 | 소대 단위 기동 명령 | 3개 소대 편제 |
| **분대장** | 9 | 5 | 분대 전술 행동, 병사 지시 | 소대당 3분대 |
| **병사** | 가변 | 6 | 개별 전투·이동 | 분대당 다수 |
| **무전병** | 2 | 4 | 통신망 유지·채널 관리·도청 방어 | 통신 핵심 역할 |
| **의무병** | 2 | 4 | 부상자 처치·후송 | KCTC 의무 체계 |
| **공병** | 1 | 4 | 지뢰 제거·교량 파괴·복구 | KCTC 공병 편제 |
| **보급관** | 1 | 4 | 5대 자원 보급 차량 운용 | KCTC 보급 체계 |
| **화기소대장** | 1 | 4 | 박격포·대전차 화기 운용 | 직할 화기소대 |

> Red Force도 동일 구조 편성. 단, 지형 숙지 보너스 +1 적용 (대항군 고증).

### 5-2. 개인 스탯 시스템

> 고증 근거: KCTC AAR에서 체력 우수 장병이 피로 상황에서도 임무 완수한 사례 반복 기록

각 플레이어는 매치 시작 전 포인트(총 10점)를 아래 스탯에 자유 배분:

| 스탯 | 기본값 | 효과 |
|------|-------|------|
| **체력 (Endurance)** | 2 | 피로도·배고픔·수면 감소 속도 저항 — 높을수록 오래 버팀 |
| **상황판단 (Judgment)** | 2 | 통신 오류 발생 시 오청 내용을 부분 감지할 확률 |
| **사격 (Accuracy)** | 2 | 기본 명중률 보정 |
| **이동력 (Mobility)** | 2 | 지형 이동 비용 감소 |
| **통신 (Comms)** | 2 | 통신 범위 보정, 도청 저항 |

**설계 의도**: 체력 스탯이 높은 플레이어는 72시간 연속작전 상황에서도 전투력 유지. 체력 낮은 플레이어는 일찍 피로 패널티 도달. 이는 실제 KCTC에서 체력이 전투 지속성을 결정하는 고증을 게임 메커니즘으로 구현.

## 6. 생존 스탯 시스템

### 6-1. 3대 생존 스탯

| 스탯 | 최대값 | 감소 방식 | 고갈 효과 |
|------|-------|---------|---------|
| **배고픔 (Hunger)** | 100 | 매 턴 -3 (기본) | 50% 이하: 이동력 -1 / 0%: AP -2 |
| **피로도 (Fatigue)** | 100 | 매 턴 -2 (기본), 전투·이동 시 추가 -3 | 50% 이하: 명중률 -10%, 오청 +10% / 0%: AP -3, 오청 +25% |
| **수면 (Sleep)** | 100 | 야간 페이즈 미수면 시 -20 | 50% 이하: 판단 스탯 -1, 시야 -1 / 0%: 행동불능 위험 |

### 6-2. 체력 스탯에 따른 내성 차이

```
체력 스탯 1점당 생존 스탯 감소 속도 -10%

예시:
- 체력 2 (기본): 배고픔 매 턴 -3
- 체력 4:        배고픔 매 턴 -2.4 (약 20% 느리게 감소)
- 체력 6:        배고픔 매 턴 -1.8 (약 40% 느리게 감소)

→ 체력에 집중 투자한 플레이어가 장기전에서 유리
→ 사격·이동력에 투자한 플레이어는 초반 전투는 강하지만 후반 스태미나 부족
```

### 6-3. 회복 수단

| 방법 | 회복량 | 조건 |
|------|-------|------|
| 전투식량 수령 | 배고픔 +40 | 보급관이 보급 실행 |
| 진지 내 대기 (1턴) | 피로도 +15 | 전투·이동 없는 턴 |
| 수면 (야간 페이즈) | 수면 +60, 피로도 +30 | 야간 페이즈 중 수면 행동 선택 |
| 의무병 처치 | 피로도 +20 | 의무병 인접 시 |

### 6-4. 야간 페이즈 수면 딜레마

> KCTC 고증: 야간 경계와 수면 확보는 항상 충돌

- 수면을 취하면 다음 턴 전투력 유지, 단 해당 턴 경계 불가
- 경계를 서면 야간 침투 방어 가능, 단 피로 누적
- 중대장이 경계 조 편성을 명령으로 지시해야 함 → 지휘 결심 메커니즘

## 7. 통신 오류 시스템

### 7-1. 통신 오류 5종

| 오류 유형 | 발생 조건 | 게임 효과 |
|---------|---------|---------|
| **오청 (Mishear)** | 통신 품질 70% 이하, 피로 누적 | 명령 텍스트 일부 변형 수신 |
| **지연 (Delay)** | 통신 품질 50% 이하 | 명령이 1~2턴 후 도달 |
| **두절 (Blackout)** | 계곡·배터리 0%·지휘소 파괴 | 명령 미전달, 수신자 모름 |
| **도청 (Intercept)** | 개활지 무전, Red Force 감청 장비 | 적팀이 명령 원문 열람 |
| **위장 삽입 (Spoof)** | 적 위장 요원 인접 성공 | 가짜 명령이 아군 채팅창에 삽입 |

### 7-2. 통신 품질 결정 요인

| 요인 | 효과 |
|------|------|
| 지형 (계곡·역사면) | 음영 구역 — 두절 또는 왜곡 |
| 무전병 부상 | 오청 확률 +30% |
| 배터리 잔량 50% 이하 | 왜곡 시작 |
| 배터리 0% | 완전 두절 |
| 피로도 0% | 오청 +25% |
| 거리 초과 | 최대 통신 거리 초과 시 두절 |
| 중계소 설치 | 음영 구역 해소, 거리 연장 |
| 고지 점령 | 통신 범위 +1 |

### 7-3. 도청 시스템 3종

**A. 무전기 탈취 도청** (고증: KCTC 수색대원 항복 시 무전기 피탈 사례)
- 트리거: 무전병 또는 지휘관 즉사 판정
- 효과: 해당 유닛 위치에 무전기 아이콘 생성
- Red Force 해당 타일 이동 시 획득 → 3턴간 Blue Force 채팅 열람
- 카운터: 아군이 시신 타일 이동 시 무전기 파괴 가능

**B. 주파수 감청** (Red Force 무전병 특수 행동)
- 조건: 감청 장비 아이템 + 2AP 소모
- 효과: 반경 내 Blue Force 채팅 노이즈 섞인 형태로 1턴 열람
- 완전 해독: 추가 1AP → 원문 해독 (총 3AP)
- 카운터: Blue Force 무전병 채널 변경 시 즉시 차단

**C. 위장 명령 삽입** (고증: 대항군 위장 침투 사례)
- 조건: Red Force 위장 요원이 아군 2타일 이내 잠입 성공
- 효과: Blue Force 채팅창에 가짜 명령 1회 삽입
- 카운터: 수신 확인 코드 입력 시 위장 탐지
- 탐지 실패 시: Blue Force 유닛이 가짜 명령대로 행동

### 7-4. 통신 채널 관리

| 채널 | 용도 | 도청 시 피해 |
|------|------|-----------|
| 지휘 채널 | 중대장↔소대장 명령 | 작전 전체 노출 |
| 화력 채널 | 포병 지원 요청 | 화력 지원 역이용 |
| 의무 채널 | 부상자 후송 | 아군 피해 현황 파악 |
| 보급 채널 | 보급 차량 경로 | 보급선 차단 가능 |

채널 교체: 무전병 1AP 소모 → 도청 즉시 차단, 단 아군 1턴 통신 두절

## 8. 교전 판정 시스템 (MILES 고증)

### 8-1. 판정 흐름

```
사격 선언
   └─ 사거리 체크 (무기별 최대 사거리)
   └─ 시야 체크 (포그 오브 워 + 통신 음영 + 야간 여부)
   └─ 명중 판정
       = 기본 명중률
       + 지형 보정
       - 피로도 패널티
       - 배고픔 패널티
       + 사격 스탯 보정
       └─ 명중 → 사상 판정 (즉사/부상/경상)
       └─ 빗나감 → 소음 발생 (위치 노출 여부 판정)
```

### 8-2. 무기별 스펙

| 무기 | 사거리 | 기본 명중률 | 피해 | 특이사항 |
|------|--------|-----------|------|---------|
| 소총 | 5헥스 | 60% | 즉사·부상 | — |
| 기관총 | 8헥스 | 50% | 즉사·부상 | 1헥스 억제 효과 |
| 박격포 | 15헥스 | 40% | 범위 3헥스 | 좌표 보고 방식 |
| 대전차화기 | 10헥스 | 70% | 차량 즉파 | 인원 관통 불가 |
| 수류탄 | 2헥스 | — | 범위 2헥스 | 살상/파편 반경 구분 |
| 포병 지원 | 30헥스+ | 35% | 범위 5헥스 | 중대장 요청, 3턴 딜레이 |

### 8-3. 사상 판정 후 처리 (KCTC 고증)

| 판정 | 즉각 효과 | 통신 영향 | 후속 처리 |
|------|---------|---------|---------|
| 즉사 | 전투 이탈 | 무전기 탈취 가능 | 인수인계 발동 |
| 부상(행동불능) | AP 0, 이동 불가 | 오청 확률 +30% | 의무병 후송 → 복귀 |
| 경상 | AP -2 | 오청 확률 +15% | 의무병 처치 → 정상 복귀 |
| 차량 피격 | 기동 불가 | — | 승무원 생존 판정 별도 |

## 9. 보급 시스템

### 9-1. 5대 보급 자원

| 자원 | 고갈 효과 | 보급 수단 | 생존 스탯 연동 |
|------|---------|---------|-------------|
| 탄약 | 해당 화기 사격 불가 | 보급 차량 | — |
| 유류 | 차량 이동 불가 | 유류 차량 | — |
| 식수 | 3턴 후 전 인원 배고픔 감소 가속 | 급수차·급수대 | 배고픔 스탯 연동 |
| 의무 물자 | 부상자 처치 불가 → 2턴 후 전사 | 의무 차량 | — |
| 통신 배터리 | 무전기 불통 | 보급 차량 | 피로도 오청 패널티 가속 |

### 9-2. 보급선 차단

- 보급 차량 경로 상 적 유닛 → 이동 불가
- 차량 피격·파괴 → 해당 물자 전량 소멸
- 보급관이 우선순위 지정 가능 (어느 자원부터 전방 투입할지)
- 중대장이 보급 우선순위 명령으로 지시 가능

### 9-3. 급수대 점령 메커니즘

- 맵 상 2~3곳 고정 위치
- 적 점령 시 해당 구역 아군 식수 차단
- 탈환 또는 정수장비(공병 설치)로 대체 가능

## 10. 특수 게임 이벤트 (랜덤·조건부)

| 이벤트 | 발생 조건 | 효과 | 학습 포인트 |
|--------|---------|------|-----------|
| **포격 좌표 오류** | 포격 요청 시 통신 품질 50% 이하 | 좌표 ±2헥스 랜덤 이동 | 통신 확인 후 포격 요청 |
| **배터리 위기** | 배터리 20% 도달 | 매 무전마다 10% 완전 두절 확률 | 배터리 관리, 필수 무전 우선순위 |
| **인수인계 긴급 발동** | 지휘관 즉사 | 30초 내 승계 미선언 시 지휘 두절 | 전 인원 지휘관 역할 숙지 |
| **위치 노출 실수** | 야간 이동·사격 시 소음 발생 | 적이 다음 턴 포격 요청 가능 | 야간 은엄폐·소음 관리 |
| **내부자 이벤트** | 매치 시작 시 Red Force가 Blue Force 1명에게 DM | 수락 시 아군 채팅 일부 적에게 전달 가능, 탐지 시 강제 격리 | 작전보안(OPSEC) |
| **장비 고장** | 배터리 5% 이하 연속 사용 | 무전기 완전 고장, 교체 전까지 통신 불가 | 장비 관리 중요성 |

## 11. 사망 및 인수인계 시스템 (KCTC 실제 방식)

### 11-1. 지휘관 사망 처리

```
중대장 사망
   └─ 부중대장 즉시 승계 (30초 이내 인수인계 선언)
       └─ 30초 초과 → 중대 전체 독립행동 모드
       └─ 부중대장도 사망 → 선임 소대장 승계

소대장 사망
   └─ 선임 분대장 즉시 승계
       └─ 미승계 → 소대 분대별 개별행동만 가능

무전병 사망
   └─ 해당 채널 즉시 두절
       └─ 다른 무전병이 채널 인수 가능 (1AP 소모)
       └─ 무전병 전원 사망 → 통신 완전 두절
```

### 11-2. 사망 플레이어 처리

- 즉사 판정 → 해당 플레이어 **관전 모드 전환**
- 관전 모드에서 AAR 데이터 실시간 수집 가능
- 인수인계로 역할 승계된 경우 → 승계한 플레이어가 두 역할 동시 수행 (AP 합산)
- 부상(행동불능) 상태 → 의무병 후송 성공 시 2턴 후 복귀

- 전 인원 피로도 0% 도달 → 전투 지속 불가 항복 처리
- 식수 고갈 5턴 지속 → 전 인원 AP -3 (사실상 행동 불능)
